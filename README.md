# Projeto SQL - Sistema de Análise de Pedidos em Banco de Dados SQL

## Descrição

Projeto desenvolvido durante o curso da EBAC com foco em consultas avançadas em SQL. O objetivo foi integrar dados de múltiplas tabelas e criar funções para análise de pedidos em um banco de dados de restaurante.

## Funcionalidades

- Criação de VIEW com JOIN entre tabelas
- Cálculo do total de pedidos
- Uso do comando EXPLAIN para análise de consultas
- Função para buscar ingredientes por produto
- Função para comparar pedido com a média geral

## Tecnologias utilizadas

- SQL
- Banco de Dados Relacional
